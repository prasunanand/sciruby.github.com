Windows Install Instructions

None of the developers use Windows, so we cannot guarantee that these instructions will work. Please join our listserv if you’re a Windows user, and we’ll be happy to help troubleshoot.

Green Shoes

You need to start by following the instructions for installing Green Shoes on Windows 7.

NMatrix

ATLAS and LAPACK are required. Detailed instructions for installing these packages are available from SciPy.

gem install nmatrix

SciRuby

gem install sciruby
gem install gtksourceview2 # optional, for code editor
gem install statsample-optimization # optional: rb-gsl, statistics2
